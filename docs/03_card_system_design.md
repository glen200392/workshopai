# 03 — 字卡系統設計邏輯

## 設計原則

字卡不是破冰工具，是**認知錨點**。

學員在報到時選擇的字卡，驅動整個工作坊的個人化體驗：
1. AI 夥伴的角色扮演練習對手
2. 討論題目的切入角度  
3. 視覺化概念圖的重點
4. 群體洞察的分析維度

---

## 12 張字卡完整定義

### 維度 A：心理立場（此刻我在哪裡？）

| 字卡 | Emoji | 副標 | roleHint（AI 扮演的練習對手）|
|---|---|---|---|
| 觀望中 | 🔭 | 先看看再說 | 催促轉型的主管 |
| 已在用了 | ⚡ | 走在大家前面 | 質疑 AI 效果的同事 |
| 半信半疑 | ⚖️ | 需要更多說服 | 過度樂觀的顧問 |
| 被推著走 | 🌊 | 還沒準備好 | 同樣不情願的同事 |

**設計邏輯**：揭露學員對 AI 轉型的初始心理定位。這個定位是後續角色扮演練習的起點——讓學員練習在自己的心理位置上，如何與不同立場的人對話。

### 維度 B：工作現實（我最在意的日常痛點）

| 字卡 | Emoji | 副標 | 討論聚焦方向 |
|---|---|---|---|
| 文件海 | 📄 | 表單永遠填不完 | 流程自動化、RPA |
| 人心難測 | 💬 | 溝通協調最燒腦 | AI 輔助溝通、情緒分析 |
| 數字壓力 | 📊 | 成本效益要說清楚 | 績效分析、ROI 計算 |
| 規則叢林 | 🏛️ | 合規讓我頭很大 | 合規自動化、法規追蹤 |

**設計邏輯**：具體錨定 HR/總務的日常場景。「AI 轉型」太抽象，但「文件海」讓學員立刻聯想到具體的表單痛苦，對話立刻變得真實。

### 維度 C：期待與恐懼（我最深的感受）

| 字卡 | Emoji | 副標 | 反思引導方向 |
|---|---|---|---|
| 解放重複 | 🔓 | 把時間留給重要的事 | 「解放後，你想做什麼？」|
| 飯碗問題 | 🍚 | 說出口需要勇氣 | 「如何表達恐懼而不被標籤」|
| 品質提升 | ✨ | 做更好的決策與服務 | 「什麼是更好的決策品質？」|
| 誰來負責 | ❓ | AI 出錯怎麼算 | AI 治理與責任框架 |

**設計邏輯**：製造群體張力。「解放重複」vs「飯碗問題」——這個分裂本身就是今天最重要的討論。讓大家看到房間裡有兩種人，然後對話。

## 字卡連動流程圖

```
學員選擇字卡
       │
       ├──→ memory.profile.roleHint 寫入
       │       ↓
       │    角色扮演時 AI 自動扮演對應角色
       │
       ├──→ buildSystemPrompt() 注入字卡背景
       │       ↓
       │    每次對話都帶入學員的字卡脈絡
       │
       ├──→ pickCounts 統計
       │       ↓
       │    講師端熱圖顯示群體分佈
       │    投影大螢幕即時更新
       │
       └──→ AI 群體洞察分析
               ↓
            「今天 70% 選了 B 維度，代表...」
```

## 如何修改字卡設計

字卡定義存在兩個地方：

**1. `frontend/student.html` 的 `S.cards` 陣列**（前端 demo 用）
**2. `frontend/instructor.html` 的「報到字卡設定」Tab**（正式使用，寫入 sessions.config）

修改字卡時，兩個地方都要更新，或者把 `S.cards` 改為從 `S.sessionConfig.checkin_cards` 讀取。

**建議的改法**（P1 任務）：
```javascript
// 在 joinSession() 取得 session config 後
S.cards = S.sessionConfig?.checkin_cards || defaultCards;
// 這樣講師端修改字卡後，學員端自動反映
```

## 不同工作坊主題的字卡範本

### 製造業數位轉型
```
維度 A：技術接受度
  🔬 測試中、🏭 生產優先、📐 規格至上、⏰ 時間壓力

維度 B：現場現實
  🔧 設備整合、📊 良率管控、👷 技術斷層、🔄 換線頻繁

維度 C：期待恐懼
  🚀 效率飛躍、🤝 人機共存、💰 投資回報、🔒 資料安全
```

### 領導力發展工作坊
```
維度 A：目前領導風格
  🎯 目標導向、🤗 關係導向、⚖️ 平衡型、🔍 分析型

維度 B：最大挑戰
  💬 跨部門溝通、🌱 人才培育、🔥 危機處理、📊 績效管理

維度 C：期待方向
  🦅 擴大影響力、🌊 更有彈性、💡 創新突破、🏗️ 打好基礎
```
